# City Weather
City Weather Dashboard

## Features
- Search for a city's weather
- See the current weather
- See the five day forecast
- Search for a new city
- Previous searches persist

## Technology Used

Materialize
https://materializecss.com/

and JavaScript, HTML, and CSS
